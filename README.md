# Vtol Camera Project

## About The Project
This is the the project of a uav equipped with a gimbal camera.

## Built With

### Environment

* Ubuntu 18.04
* ROS Melodic

### Simulatior

* Gazebo 9.19.0
* PX4-Autopilot

### Recognition

* Darknet_ros - YOLOv3
